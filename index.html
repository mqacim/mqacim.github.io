  • Natural Language Processing Engineer - Quarrio

    September 2016 – Present

    Working on the development of Deterministic Semantic AI utilizing multi-level parsers and grammars that understand language nuances, from general English to industry-specific jargon. AI system enables anyone, regardless of technical background, to interact with data seamlessly. Unlike probabilistic AI models that guess the best possible answers, deterministic approach ensures precise, contextually accurate responses every time.
    Natural Language Processing
    AI system understands questions asked on structured data in plain English language. Question as understood by the system is provided to the user in plain English as well, making validation easy.
    Cognitive Language Processing
    AI system navigates through an abstraction layer that identifies and accesses relevant data sources. It generates precise query statements, retrieves information, and presents it in an easy-to-understand format.
    Contextual Understanding
    AI system understands context of a question. For a given question, it establishes references with previously asked questions to provide response using the context in which that question was asked. AI system engages with user to get additional information before answering a question in case it cannot completely interpret the question.
